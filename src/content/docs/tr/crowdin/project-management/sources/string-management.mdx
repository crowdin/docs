You can view all the project's strings and manage strings settings via **Sources > Strings**.

## Dizgilerin Görünürlüğünü Değiştirme

Eğer bazı dizgiler çevrilmesi gerekmeyen veriler içeriyorsa (örn. yer tutucular, diğer teknik varlıklar), bunları çevirmenlerden gizleyebilirsiniz. Bunu yapmak için:

<Steps>
  1. Open your project and go to **Sources > Strings**.
  2. Select the necessary strings by clicking the checkbox next to each one.
  3. **Gizle**’ye tıklayın. <Image src={stringHide} alt="Görünürlüğü değiştirme" />
</Steps>

## Dizgileri Süzme

By default, all source strings of the project are displayed in **Sources > Strings**. Eğer gerekirse, mevcut süzgeç seçeneklerini (Sorunlar: Tümünü Göster, Çözülmemiş Sorunları Olan, Çözülmemiş Sorunları Olmayan; Yorumlar: Tümünü Göster, Yorumları Olan, Yorumları Olmayan; Ekran Görüntüleri: Tümünü Göster, Ekran Görüntüleri Olan, Ekran Görüntüleri Olmayan; Görünürlük: Tümünü Göster, Gizli ve Görünür) ve proje etiketlerini kullanarak dizgileri süzebilirsiniz.

<Image src={stringFilter} alt="Dizgileri Süzün" />

## Adding Strings

Crowdin’deki projelerinize doğrudan dizgileri ekleyebilirsiniz.

To add a new string, follow these steps:

<Tabs syncKey="projectType">
  <TabItem label="Dosya tabanlı proje">
    <Steps>
      1. Open your project and go to **Sources > Strings**.
      2. Click **Add String**.
      3. In the appeared dialog, enter the necessary details for the string:
         * *String* &ndash; specify the text that needs translation.
         * *(Optional)* *Plurals* &ndash; select this option to add plural forms and enter the text for each form based on the project’s source language requirements.
         * *Identifier* &ndash; specify a unique key for the string, often used for referencing the translated text in the application.
         * *(Optional)* *Context* &ndash; add additional information to help translators understand the intended meaning.
         * *(Optional)* [*Labels*](#labels) &ndash; specify labels to organize strings or provide extra context.
         * *(Optional)* *Maximum Translation Length* &ndash; set a character limit for the translation.
         * *(Optional)* File-specific identifiers &ndash; specify unique keys for each file format when adding the string to multiple files. This ensures that the string meets the different identifier requirements based on the file formats it will be used in.
         * *Files* &ndash; choose the source files where this string should be added.
      4. Click **Add** to add the new string to your project.
    </Steps>
    <Aside>
      For file-based projects, this feature is available only for file formats that support [online editing](#formats-that-support-online-editing).
    </Aside>
  </TabItem>
  <TabItem label="Dizgi tabanlı proje">
    <Steps>
      1. Open your project and go to the **Strings** tab.
      2. Click **Add String**.
      3. In the appeared dialog, enter the necessary details for the string:
         * *Branches* &ndash; choose the branches where this string should be added.
         * *String* &ndash; specify the text that needs translation. You can leave this field empty to create placeholder strings for future content or development purposes.
         * *(Optional)* *Plurals* &ndash; select this option to add plural forms and enter the text for each form based on the project’s source language requirements.
         * *Identifier* &ndash; specify a unique key for the string, often used for referencing the translated text in the application.
         * *(Optional)* *Context* &ndash; add additional information to help translators understand the intended meaning.
         * *(Optional)* [*Labels*](#labels) &ndash; specify labels to organize strings or provide extra context.
         * *(Optional)* *Maximum Translation Length* &ndash; set a character limit for the translation.
         * *(Optional)* File-specific identifiers &ndash; specify unique keys for each branch when adding the string to multiple branches. This ensures that the string meets the different identifier requirements based on the file formats it will be used in.
      4. Click **Add** to add the new string to your project.
    </Steps>
  </TabItem>
</Tabs>

<ReadMore>

</ReadMore>

<Aside type="caution" title="Sınırlamalar">
  The maximum identifier length is 512 characters.
</Aside>

## Dizgi Düzenleme

You can add context to the string, add or remove labels, set the max. length of the translation. Once the translation limits are exceeded, the system notifies the contributor that the translation should be shorter.

<Image src={stringEdit} alt="Dizgi Düzenleme" class="width-lg" />

Birden çok kaynak dizgide görünen belirli bir kelimeyi veya ifadeyi düzenlemek için **Kaynaklar > Dizgiler**’de **Bul ve Değiştir** özelliğini kullanabilirsiniz.

<Aside type="caution">
  When editing the identifier of a string contained in a group of strings (string array for Android XML, segments for XLIFF), the existing translations will be removed for all group strings but the edited one.
</Aside>

### Etiketler

Dizgilere bağlam eklemenin veya bunları belirli konulara göre düzenlemenin kolay bir yolu için projenizde etiketleri kullanın. Etiketler aynı zamanda çeviri ve düzeltme okuması görevleri oluştururken veya [Gelişmiş Süzgeç](/online-editor/#gelişmiş-süzgeç) yardımıyla Düzenleyicide belirli dizgileri ararken de faydalıdır.

<LinkCard title="Proje Etiketlerini Yönetme" description="Learn how to make your Crowdin project labels." href="/project-settings/labels/" />

Dizgilere etiketleri ekleyebileceğiniz birkaç olası yol vardır.

##### Add labels to one string at a time via the *Edit String* dialog

<Steps>
  1. Listedeki dizgide **Düzenle**’ye tıklayın.
  2. Select needed labels in the *Labels* field. <Image src={labelsAddingToOneString} alt="Etiketler Ekleyin" class="width-lg" />
  3. **Kaydet**’e tıklayın.
</Steps>

##### Add labels to multiple strings via the *Manage Strings Labels* dialog

<Steps>
  1. Select a few strings you want to add the same labels to.
  2. Click **Labels**.
  3. *Eklenecek etiketler* alanında gerekli etiketleri seçin. <Image src={labelsAddingToMultipleStrings} alt="Etiketler Ekleyin" class="width-lg" />
  4. **Kaydet**’e tıklayın.
</Steps>
