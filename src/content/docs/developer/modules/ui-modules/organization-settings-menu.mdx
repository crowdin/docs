The module allows the creation of a new section in the Organization Settings page.

<Image src={orgSettingsMenu} alt="Organization Settings Menu Module" />

## Access

You can grant access to this module to one of the following user categories:

* Only organization admins
* Selected organization admins

## Structure

```json title="manifest.json"
{
  "modules": {
    "organization-settings-menu": [
      {
        "key": "your-settings-module-key",
        "name": "Settings Module name",
        "url": "/organization-settings-page",
        "icon": "/images/settings-icon.png"
      }
    ]
  }
}
```
