Crowdin Editor uses JSON-based files to manage its visual themes. This article outlines the key concepts for designers to create and distribute custom themes.

## Theme File Structure

A complete Crowdin theme file contains the following top-level properties:

* `themeMode` &ndash; accepts `light` or `dark` for base editor styling.
* `primaryAccent` &ndash; main accent color used throughout the interface.
* `base` &ndash; a collection of properties controlling various interface elements (backgrounds, font colors, string status colors, highlights).
* `accents` &ndash; colors used for notifications (info, danger, warning, success messages).
* `button` &ndash; provides styling options for different button types.

## Important Notes

* **Color Values** &ndash; all color properties require valid CSS color values (HEX, RGB, RGBA).
* **Complex Backgrounds** &ndash; properties `base.commonUi.headerBackground`, `base.commonUi.subHeaderBackground`, `base.commonUi.mainContentBackground` and `base.commonUi.mainMenuBackground` accept gradients, images, and other valid CSS background values.
* **Minimal Themes** &ndash; it's not mandatory to define every property. Crowdin automatically calculates missing values based on the provided settings.

### Development and Testing

During the theme development process, we recommend using the [Theme Builder](https://store.crowdin.com/theme-builder) app, with the help of which designers can easily test and preview their themes in real time.

```json
  "themeMode": "dark",
  "primaryAccent": "#f9d9c8",
  "base": {
    "baseBackground": "#2d2b52",
```

```json
    }
  }
}
```

## Theme Publishing

Custom themes should be submitted to the Crowdin Store to become available for installation by Crowdin users.

To publish a theme, follow these steps:

<Steps>
  1. [Contact Support Team](https://crowdin.com/contacts): We will handle the review and publishing process 24x7.
  1. Provide the following information:
      * Author name
      * Theme name
      * Description
      * Screenshot of the theme in use  
</Steps>
