Use labels in your project to easily organize resources (e.g., screenshots, strings, etc.) by specific topics. Once you add labels, you can filter items by those labels. If you already use labels for your source strings, you can use the same labels for screenshots.

## Managing Project Labels

To create, edit, or remove project labels, follow these steps:

<Steps>
  1. Open your project **Settings** tab.
  1. Select the <Icon name="mdi:label" class="inline-icon" /> **Labels** section.
  1. In the appeared screen, add new labels and edit or remove existing ones.
</Steps>

<Image src={projectLabels} alt="Manage project labels" class="width-xl" />

To edit or remove a label, click the <Icon name="mdi:dots-horizontal" class="inline-icon" /> icon next to the label you want to edit or remove and select the appropriate action.

<Aside type="caution">
  Labels are case insensitive (e.g., `Label` and `label` are treated as identical).
</Aside>
