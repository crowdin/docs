In the **Machine Translation** section, you can manage the following settings:

* **Show machine translation suggestions** &ndash; enable MT suggestions from machine translation engines such as Microsoft Translator, Google Translate, and others to be displayed in the Editor.

<Aside>
  The project owner should [configure machine translation engines](/machine-translation/) before using them.
</Aside>
