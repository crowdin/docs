Use **Conversations** to communicate with other project members. Create one-to-one or group conversations, each with a subject to help you easily navigate between chats.

<Image src={all} alt="All conversations" />

## Creating a New Conversation

To create a new conversation, follow these steps:

<Steps>
  1. Open **Conversations**.
  2. Click <Icon name="mdi:plus" class="inline-icon" />**Create Conversation**.
  3. Filter users by:
      - project, role, language
      - recent contacts – users you already have conversations with
  4. Select the users you want to message.
  5. Click **Go**.
</Steps>

<Image src={createNewSearch} alt="Create new conversation" />

## Managing Conversations

To manage a conversation, click <Icon name="mdi:cog" class="inline-icon" /> and choose one of the following options:

- *Change Subject* – displayed to all participants to help organize conversations.
- *Mute Conversation* – stop receiving notifications about new messages.
- *Add Users* – participants from common projects. A new conversation will be created with the new user set.

<Image src={dialogOptions} alt="Dialog options" />

<Aside type="caution" title="Limitations">
  Deleting conversations is currently not supported. This functionality is planned for future updates.
</Aside>

## Messages

To manage a message, hover over it, click <Icon name="mdi:dots-horizontal" class="inline-icon" /> and select the needed option. The following actions are available:

- *Share* – forward the message:
  - in the same conversation
  - to another conversation
  - to a user you’ve previously messaged
- *Mark Unread* – mark the message as unread to return to it later.
- *Edit* – available for your own messages.
- *Delete* – available for your own messages.
- *Report Spam* – available for messages from others.

<Image src={messageOptions} alt="Message options" />

## Unread Messages

Unread messages are indicated by a counter displayed:

- next to each conversation’s subject, showing the number of unread messages in that conversation
- at the top of the page next to the *Messages* icon, showing the total number of unread messages across all conversations
