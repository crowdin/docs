聊天模块允许在 Crowdin 或 Crowdin Enterprise 界面的右侧面板中创建聊天对话框。 可通过标题栏中用户头像旁带有应用程序徽标的按钮来打开或关闭该面板。

## 访问权限

您可以将此模块的访问权限授予以下用户类别之一：

对于 Crowdin：

* 仅限我自己（即项目所有者）
* 所有项目成员
* 选定用户

对于 Crowdin Enterprise：

* 仅限组织管理员
* 组织项目中的所有用户
* 选定用户

## 结构

```json title="manifest.json"
{
  "modules": {
    "chat": [
      {
        "key": "your-module-key",
        "name": "Module name",
        "description": "Module description",
        "url": "/chat-page"
      }
    ]
  }
}
```
