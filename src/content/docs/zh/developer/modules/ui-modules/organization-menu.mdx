该模块允许在工作区主页的左侧面板中创建新板块。

<Image src={orgMenu} alt="组织菜单模块" />

## 访问权限

您可以将此模块的访问权限授予以下用户类别之一：

* 仅限组织管理员
* 组织项目中的所有用户
* 选定用户

## 结构

```json title="manifest.json"
{
  "modules": {
    "organization-menu": [
      {
        "key": "your-module-key",
        "name": "Module name",
        "url": "/organization-page",
        "icon": "/images/icon.png"
      }
    ]
  }
}
```
