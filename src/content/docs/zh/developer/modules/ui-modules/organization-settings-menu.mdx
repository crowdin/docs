该模块允许在组织设置页面中创建新板块。

<Image src={orgSettingsMenu} alt="组织设置菜单模块" />

## 访问权限

您可以将此模块的访问权限授予以下用户类别之一：

* 仅限组织管理员
* 已选择的组织管理员

## 结构

```json title="manifest.json"
{
  "modules": {
    "organization-settings-menu": [
      {
        "key": "your-settings-module-key",
        "name": "Settings Module name",
        "url": "/organization-settings-page",
        "icon": "/images/settings-icon.png"
      }
    ]
  }
}
```
