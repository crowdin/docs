模块允许您在编辑器右侧面板中创建额外的标签页。 当在您的 Crowdin 应用程序中使用此模块时，您可以选择编辑器显示额外标签页的模式。

<Image src={editorRP} alt="编辑器右侧面板模块" />

## 访问权限

您可以将此模块的访问权限授予以下用户类别之一：

对于 Crowdin：

* 仅限我自己（即项目所有者）
* 本人、项目管理员和开发者
* 所有项目成员
* 选定用户

对于 Crowdin Enterprise：

* 仅限组织管理员
* 组织管理员、项目管理员和开发者
* 组织项目中的所有用户
* 选定用户

## 结构

```json title="manifest.json"
{
  "modules": {
    "editor-right-panel": [
      {
        "key": "your-module-key",
        "name": "Module name",
        "modes": [
          "translate"
        ],
        "supportsMultipleStrings": false,
        "url": "/editor-page"
      }
    ]
  }
}
```
