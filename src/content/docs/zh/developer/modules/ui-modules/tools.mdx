工具模块允许在 Crowdin 或 Crowdin Enterprise 项目中创建并插入新的工具页面。 您可以在 **Tools** 部分找到它。

<Image src={tools} alt="工具模块" />

## 访问权限

您可以将此模块的访问权限授予以下用户类别之一：

对于 Crowdin：

* 仅限我自己（即项目所有者）
* 所有项目成员
* 选定用户

对于 Crowdin Enterprise：

* 仅限组织管理员
* 组织项目中的所有用户
* 选定用户

## 结构

```json title="manifest.json"
{
  "modules": {
    "project-tools": [
      {
        "key": "your-module-key",
        "name": "模块名称",
        "description": "模块描述",
        "logo": "/logo.png",
        "url": "/tools-page"
      }
    ]
  }
}
```
