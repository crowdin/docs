该模块允许在用户的个人资料菜单中创建新项目。

<Image src={profileMenu} alt="个人资料菜单模块" />

<Aside>
  此模块仅适用于 Crowdin。
</Aside>

## 访问权限

您可以将此模块的访问权限授予以下用户类别之一：

* 仅限我自己（即个人资料所有者）

## 结构

```json title="manifest.json"
{
  "modules": {
    "profile-resources-menu": [
      {
        "key": "your-module-key",
        "name": "Module name",
        "url": "/resource-page",
        "icon": "/images/icon.png",
        "environments": "crowdin"
      }
    ]
  }
}
```
