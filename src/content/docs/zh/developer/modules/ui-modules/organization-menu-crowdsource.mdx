该模块允许在 Crowdin Enterprise 的众包公共页面上创建额外的标签。 要使用它，请确保您至少有一个包含众包工作流步骤的项目，并且该项目已发布在众包设置页面上。

## 访问权限

您可以将此模块的访问权限授予以下用户类别之一：

* 仅限组织管理员
* 组织项目中的所有用户
* 选定用户
* 访客（未经身份验证的用户）

## 结构

```json title="manifest.json"
{
  "modules": {
    "organization-menu-crowdsource": [
      {
        "key": "your-module-key",
        "name": "模块名称",
        "url": "/crowdsource-page"
      }
    ]
  }
}
```
