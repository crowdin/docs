该模块允许在 Crowdin Enterprise 中为项目的众包公开页面创建附加标签页。 使用前，请确保您的项目满足以下要求：

* 项目的工作流包含众包步骤。
* 该项目已在众包设置页面中发布。

## 访问权限

您可以将此模块的访问权限授予以下用户类别之一：

对于 Crowdin：

* 仅限我自己（即项目所有者）
* 所有项目成员
* 选定用户
* 访客（未经身份验证的用户）

对于 Crowdin Enterprise：

* 仅限组织管理员
* 组织项目中的所有用户
* 选定用户
* 访客（未经身份验证的用户）

## 结构

```json title="manifest.json"
{
  "modules": {
    "project-menu-crowdsource": [
      {
        "key": "your-module-key",
        "name": "Module name",
        "url": "/crowdsource-page"
      }
    ]
  }
}
```
