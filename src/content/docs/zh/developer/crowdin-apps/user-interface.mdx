UI 是任何应用程序的必要部分。 吸引人的外观和经过深思熟虑的直观界面会让您的 Crowdin 应用程序更加用户友好，因而吸引更多用户使用它。

我们推荐使用框架来让您的应用程序看起来更好且更加融入 Crowdin UI。

以下是您可以用于 Crowdin 应用程序的框架简短列表。 当然，您可以使用任何框架进行 UI 开发。 此列表仅供参考。

  <LinkCard title="React Bootstrap" description="基于 React 构建的 Bootstrap 组件。" href="https://react-bootstrap.github.io/" target="_blank" />
