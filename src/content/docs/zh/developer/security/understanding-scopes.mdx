作用域让您可以设置所需的确切访问类型。 作用域限制了个人访问令牌、OAuth 令牌和 Crowdin 应用程序的访问范围。 它们不会授予超出用户权限范围的任何额外权限。

您在 Crowdin 上为 OAuth 应用程序指定的作用域将在授权表单上显示给用户。
