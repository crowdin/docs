当您使用私有集成工作时（例如，与自己托管的 VCS 进行集成）或您的公司网络使用防火墙时，您需要将 Crowdin 的 IP 地址和域名添加到允许列表中，以确保安全性的同时，Crowdin 正常工作。 使用 Crowdin webhooks 时，也需要采用类似的方法。

<Aside type="caution">
  以下列出的 IP 地址会定期更改。
</Aside>
