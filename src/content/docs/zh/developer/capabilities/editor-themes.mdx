Crowdin 编辑器使用基于 JSON 的文件来管理其视觉主题。 本文概述了设计师创建和分发自定义主题的关键概念。

<LinkCard title="探索现有主题" description="在 Crowdin Store 上浏览可用主题集合。" href="https://store.crowdin.com/collections/themes" target="_blank" />

## 主题文件结构

完整的 Crowdin 主题文件包含以下顶级属性：

* `themeMode` &ndash; 接受 `light` 或 `dark`，用于编辑器基础样式。
* `primaryAccent` &ndash; 整个界面中使用的主强调色。
* `base` &ndash; 一组控制各种界面元素（背景、字体颜色、字符串状态颜色、高亮）的属性。
* `accents` &ndash; 用于通知的颜色（信息、危险、警告、成功消息）。
* `button` &ndash; 为不同按钮类型提供样式选项。

## 重要说明

* **颜色值** &ndash; 所有颜色属性需要有效的 CSS 颜色值（HEX、RGB、RGBA）。
* **复杂背景** &ndash; 属性 `base.commonUi.headerBackground`、`base.commonUi.subHeaderBackground`、`base.commonUi.mainContentBackground` 和 `base.commonUi.mainMenuBackground` 接受渐变、图片及其他有效的 CSS 背景值。
* **最简主题** &ndash; 并非所有属性都必须定义。 Crowdin 会根据所提供的设置自动计算缺失的值。

### 开发与测试

在主题开发过程中，我们建议使用 [Theme Builder](https://store.crowdin.com/theme-builder) 应用，借助该应用，设计师可以轻松地实时测试和预览其主题。

```json
  "base": {
    "baseBackground": "#2d2b52",

    },
```

```json
    }
  }
}
```

## 主题发布

自定义主题应提交到 Crowdin 商店，以便 Crowdin 用户安装使用。

要发布主题，请按以下步骤操作：

<Steps>
  1. [联系支持团队](https://crowdin.com/contacts)：我们将全天候（24x7）处理审核和发布流程。
  2. 提供以下信息：
     * 作者姓名
     * 主题名称
     * 描述
     * 主题使用中的截图
</Steps>
