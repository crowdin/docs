在**机器翻译**部分，您可以管理以下设置：

* **显示机器翻译建议** &ndash; 启用后，机器翻译引擎（例如微软翻译、Google 翻译等）的翻译建议将显示在编辑器中。

<Aside>
  项目所有者应在使用机器翻译引擎前[配置机器翻译引擎](/zh/machine-translation/)。
</Aside>
