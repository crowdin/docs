为了能够上传您要本地化的内容，您首先需要创建一个项目。

## 项目可见性

在创建项目时，您可以选择首选的*项目可见性*：

<CardGrid>
  <Card title="公开项目" icon="magnifier">
    对所有人可见。 该项目将出现在 Crowdin 全局搜索结果中，并被搜索引擎收录。
    您可以在项目创建后配置协作和访问设置。
  </Card>
  <Card title="私有项目" icon="seti:lock">

  </Card>

## 项目类型

Crowdin 为您组织和管理本地化流程的方式提供了灵活性。 主要有两种项目类型：*基于文件*和*基于字符串*。 选择最适合您内容结构和翻译管理方式的类型。

### <Icon name="mdi:file-document" class="inline-icon" />基于文件的项目

基于文件的项目是 Crowdin 中的标准本地化设置，非常适合翻译应用程序和文档。 在这种项目类型中，您需要上传包含可翻译内容的源文件。 Crowdin 扫描这些文件，识别需要翻译的文本，并在项目中保留原始文件。 翻译完成后，您可以导出文件，同时保留原始结构和格式。

当导出时保留原始文件结构至关重要，以及与版本控制系统（例如 GitHub、GitLab、Bitbucket、Azure Repos）和外部数据源集成时，此项目类型是理想选择。

Crowdin 支持软件开发和文档中使用的[多种格式](/zh/supported-formats/)，包括 Android XML、iOS Strings、JSON、XLIFF、Markdown、DOCX 等。

#### 用例

* 翻译存储在结构化文件中的应用程序字符串和文档。
* 导出译文时保留原始文件夹和文件结构。
* 使源文件与代码仓库保持同步。
* 使本地化与开发和发布工作流保持一致。
* 适用于版本控制内容或结构化文档。

### <Icon name="mdi:format-list-bulleted" class="inline-icon" />基于字符串的项目

Crowdin 中基于字符串的项目旨在通过集中式字符串存储库管理动态内容。 Crowdin 不存储源文件，而是提取各个可翻译字符串并在平台内进行管理。 此设置非常适合频繁变更的内容，例如 UI 文本和其他模块化内容块。

使用这种方式，您可以直接在 Crowdin 中更新源字符串，无需重新上传文件。 翻译后的字符串可以导出为多种格式，包括 JSON、iOS Strings 或 Android XML，便于在 Web、移动端和桌面端等多个平台上重复使用。

#### 用例

* 在集中位置管理和翻译应用程序或网站 UI 文本。
* 让内容撰写者无需开发人员介入即可在线更新源字符串。
* 无需管理源文件即可翻译数据库内容。
* 避免对跨平台（例如 Android、iOS、Web）复用的字符串重复翻译。
* 以您偏好的格式导出译文，包括 JSON、iOS Strings 或 Android XML。
* 非常适合处理动态内容和频繁更新的团队。

<Aside type="caution" title="限制">
  部分连接器支持有限，某些功能在基于字符串的项目中可能不可用。
</Aside>

## 创建项目

要创建项目，请遵循以下步骤：

<Steps>
  1. [登录](https://crowdin.com/login)您的 Crowdin 账户，或[注册](https://crowdin.com/join)创建一个账户。
  2. 点击**创建项目**。 <Image src={createProject} alt="创建项目" />
  3. 输入**项目名称**。
  4. 如有需要，自定义您的**项目地址**。 这将是您项目的公开 URL。
  5. 选择**项目可见性**。 您可以在**公开项目**（对任何人可见）和**私有项目**（仅对受邀成员可见）之间进行选择。 进一步的访问控制可在创建后于[项目设置](/zh/project-settings/privacy-collaboration/)中进行配置。 <Image src={projectVisibility} alt="项目可见性" />
  6. 为您的项目选择**源语言**和**目标语言**。 源语言是要翻译的语言，目标语言是您要翻译到的语言。
  7. 选择最适合您内容和本地化工作流程的项目类型：[**基于文件的项目**](#基于文件的项目)或[**基于字符串的项目**](#基于字符串的项目)。
     <Aside type="tip">您可以手动选择目标语言，或使用**预填充**选项从您账户下的其他项目[复制目标语言](/zh/project-settings/languages/#复制目标语言)，或选择最常用的 30 种语言。 如需翻译为官方不支持的语言，您可以将其添加为[自定义语言](/zh/project-settings/languages/#添加自定义语言)。</Aside>
  8. *（可选）* 选择**安装引导应用**，以获得 Crowdin 功能的引导介绍。 项目创建后，引导应用将自动启动，帮助您完成初始设置步骤。
  9. *（可选）* 在**高级设置**下，您可以选择现有的**翻译记忆**和**词汇表**作为新项目的默认设置。 默认情况下，将为项目创建一个新的空翻译记忆和词汇表。 您可以在项目的**设置**中更改[默认翻译记忆](/zh/project-settings/translation-memories/#更改默认翻译记忆)和[默认词汇表](/zh/project-settings/glossaries/#更改默认词汇表)。
  10. 点击**创建项目**。
</Steps>

<Aside type="caution">
  项目一旦创建，项目地址和类型将无法更改。
</Aside>
