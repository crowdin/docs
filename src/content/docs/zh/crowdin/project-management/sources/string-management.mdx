您可以通过 **原文 > 字符串** 查看项目的所有字符串并管理字符串设置。

## 更改字符串可见性

如果某些字符串包含不应翻译的数据（例如占位符、其他技术实体），您可以对翻译员隐藏它们。 如需隐藏：

<Steps>
  1. 打开您的项目，并前往 **原文 > 字符串**。
  2. 点击每个字符串旁边的复选框以选择所需字符串。
  3. 单击**隐藏**。 <Image src={stringHide} alt="更改可见性" />
</Steps>

## 字符串筛选

默认情况下，项目的所有源字符串都在 **原文 > 字符串** 中显示。 如有必要，您可以使用可用的筛选器选项筛选掉字符串（问题：显示所有、有未解决的问题、无未解决的问题；评论：全部、有评论、无评论；屏幕截图：显示所有、有屏幕截图、无屏幕截图；可见性：显示所有、 仅隐藏和仅可见）和项目标记。

<Image src={stringFilter} alt="筛选字符串" />

## 添加字符串

您可以直接在 Crowdin 中向项目添加字符串。

要添加新字符串，请按照以下步骤操作：

<Tabs syncKey="projectType">
  <TabItem label="基于文件的项目">
    <Steps>
      1. 打开您的项目，并前往 **原文 > 字符串**。
      2. 点击 **添加字符串**。
      3. 在弹出的对话框中，输入字符串的必要详情：
         * *字符串* &ndash; 指定需要翻译的文本。
         * *（可选）* *复数形式* &ndash; 选择此选项以添加复数形式，并根据项目源语言要求为每种形式输入文本。
         * *标识符* &ndash; 为字符串指定唯一键，通常用于在应用程序中引用已翻译的文本。
         * *（可选）* *上下文* &ndash; 添加附加信息以帮助译员理解预期含义。
         * *（可选）* [*标记*](#标记) &ndash; 指定标记以组织字符串或提供额外上下文。
         * *（可选）* *最大翻译长度* &ndash; 为翻译设置字符限制。
         * *（可选）* 文件特定标识符 &ndash; 将字符串添加到多个文件时，为每种文件格式指定唯一键。 这可确保字符串满足其所用文件格式的不同标识符要求。
         * *文件* &ndash; 选择应添加此字符串的源文件。
      4. 点击 **添加** 将新字符串添加到您的项目。
    </Steps>
    <Aside>
      对于基于文件的项目，此功能仅适用于支持[在线编辑](#支持在线编辑的格式)的文件格式。
    </Aside>
  </TabItem>
  <TabItem label="基于字符串的项目">
    <Steps>
      1. 打开您的项目并转到 **字符串** 选项卡。
      2. 点击 **添加字符串**。
      3. 在弹出的对话框中，输入字符串的必要详情：
         * *分支* &ndash; 选择应添加此字符串的分支。
         * *字符串* &ndash; 指定需要翻译的文本。 您可以将此字段留空，以为未来的内容或开发目的创建占位字符串。
         * *（可选）* *复数形式* &ndash; 选择此选项以添加复数形式，并根据项目源语言要求为每种形式输入文本。
         * *标识符* &ndash; 为字符串指定唯一键，通常用于在应用程序中引用已翻译的文本。
         * *（可选）* *上下文* &ndash; 添加附加信息以帮助译员理解预期含义。
         * *（可选）* [*标记*](#标记) &ndash; 指定标记以组织字符串或提供额外上下文。
         * *（可选）* *最大翻译长度* &ndash; 为翻译设置字符限制。
         * *（可选）* 分支特定标识符 &ndash; 将字符串添加到多个分支时，为每个分支指定唯一键。 这可确保字符串满足其所用文件格式的不同标识符要求。
      4. 点击 **添加** 将新字符串添加到您的项目。
    </Steps>
  </TabItem>
</Tabs>

  标识符的最大长度为 512 个字符。

## 字符串编辑

您可以为字符串添加上下文、添加或删除标记、设置翻译的最大长度。 一旦超过翻译长度限制，系统将通知贡献者翻译内容应更短。

<Image src={stringEdit} alt="字符串编辑" class="width-lg" />

要编辑多个源字符串中出现的某些特定单词或短语，您可以使用**原文 > 字符串**中的**查找与替换**功能。

<Aside type="caution">
  在编辑一组字符串（Android XML 的字符串数组、XLIFF 的片段）中所含字符串的标识符时，除已编辑的字符串外，所有组字符串的现有翻译都将被删除。
</Aside>

### 标记

在项目中使用标记可以轻松地向字符串添加语境或按某些主题组织它们。 在创建翻译和校对任务，或借助[高级筛选器](/zh/online-editor/#高级筛选器)在编辑器中搜索特定字符串时，标记也很有用。

<LinkCard title="管理项目标记" description="了解如何创建 Crowdin 项目标记。" href="/zh/project-settings/labels/" />

有几种可能的方法可以向字符串添加标记。

##### 通过*编辑字符串*对话框一次向一个字符串添加标记

<Steps>
  1. 在列表中单击该字符串的**编辑**。
  2. 在 *标记* 字段中选择所需标记。 <Image src={labelsAddingToOneString} alt="添加标记" class="width-lg" />
  3. 点击**保存**。
</Steps>

##### 通过*管理字符串标记*对话框向多个字符串添加标记

<Steps>
  1. 选择要添加相同标记的几个字符串。
  2. 点击 **标签**。
  3. 在*要添加的标记*字段中选择所需标记。 <Image src={labelsAddingToMultipleStrings} alt="添加标记" class="width-lg" />
  4. 点击**保存**。
</Steps>
