使用**账户设置**页面管理您的 Crowdin 账户设置，包括个人资料详情、通知偏好、安全设置等。 要访问它，请点击右上角的个人头像，然后选择**设置**。

## 个人资料

在**个人资料**选项卡中，您可以管理您的个人详情、语言偏好、外观设置和隐私选项。

<Image src={introAccountSettings} alt="个人资料" />

### 个人头像

通过将文件拖到指定区域或从您的设备中选择文件来上传个人头像。 您可以随时移除头像。

### 基本信息

更新您的全名、用户名、电子邮件地址、公司、职位、代词以及关于您自己的简短描述。

您的电子邮件地址不公开显示，用于账户相关通知，例如密码重置、项目更新和邀请。 您还可以设置[其他登录方式](#auth-providers)以便更轻松地访问您的账户。

### 语言和地区

设置您偏好的 Crowdin 界面语言、时区和时间格式（12 小时制或 24 小时制）。 此外，指定用于显示项目目标语言的首选语言，使其在目标语言列表中优先显示。

<Image src={languageRegion} alt="语言和地区" />

### 外观

选择 Crowdin 在您的设备上的显示方式。 您可以选择浅色或深色主题，或与系统设置同步，以便在白天和夜间模式之间自动切换。

<Image src={theme} alt="颜色主题" />

### 隐私

启用**隐私个人资料**以对访问您个人资料页面的其他用户隐藏**项目**和**活动**选项卡。 启用后，访客将看到*此个人资料为私密*消息，而不是您的项目列表和近期操作。

<Image src={privateProfile} alt="该用户的个人资料为隐藏状态" />

要将您的个人资料设为私密，请按以下步骤操作：

<Steps>
  1. 打开您的**账户设置**，然后转到**个人资料**选项卡。
  2. 向下滚动至**隐私**部分。
  3. 选择**隐私个人资料**。
</Steps>

### 迁移至 Crowdin 企业版

您可以将您的 Crowdin 账户数据（包括项目、翻译记忆库和术语库）迁移到一个新的或现有的 Crowdin 企业版组织。

<ReadMore>
  进一步阅读[迁移至 Crowdin 企业版](/zh/enterprise/migrating-to-crowdin-enterprise/)。
</ReadMore>

### 删除账户

出于安全考虑，我们不能代表用户删除其账号。 您必须有权访问您的 Crowdin 账户才能将其删除。

若要删除您的账号，请按以下步骤操作：

<Steps>
  1. 打开您的**账户设置**，然后转到**个人资料**选项卡。
  2. 向下滚动至**删除账户**部分。
  3. 点击**删除账户**。
</Steps>

<Aside type="danger">
  删除账户将立即删除在您账户下创建的所有项目及所有相关数据。 已删除的账户无法恢复！
</Aside>

账户删除后，您的译文、评论、术语和投票将保留在系统中，但将显示为由**已删除的用户**添加。

## 账户

在**账户**选项卡中，您可以管理密码、使用安全密钥和通行密钥增强账户安全性、设置双重身份验证、管理已关联的身份验证提供商，以及查看活跃会话。 您还可以验证新设备，并在敏感操作期间启用 sudo 模式以提供额外安全保障。

### 密码

在**密码**部分，您可以通过点击**更改密码**并按照说明操作来更改当前密码。 如果您的账户是通过 Google 或 Facebook 等身份验证提供商创建的，您可以创建新密码以在不使用这些提供商的情况下登录。

##### 密码强度

为确保最高级别的账户安全，Crowdin 使用[熵分数](https://en.wikipedia.org/wiki/Password_strength#Entropy_as_a_measure_of_password_strength)来评估计算机猜测您密码的难度。 我们不依赖简单的规则，例如*必须包含一个数字*。 相反，我们根据长度和字符不可预测性来计算复杂度。

当您输入密码时，密码字段下方的彩色条会填充并变色，以指示密码的强度。

**密码接受策略：**

Crowdin 禁止使用被归类为**非常弱**和**弱**的密码。

如果您尝试保存具有上述评级之一的密码，系统将自动清除密码字段并显示错误消息：**请选择更安全的密码。**

请参考下表了解安全级别和视觉指示器：

| 强度     | 视觉指示器 | 状态      | 描述                                   |
| :----- | :---- | :------ | :----------------------------------- |
| **极弱** | 红色条   | **已拒绝** | **有风险。** 通常是可被立即猜到的常见字典词汇。           |
| **弱**  | 橙色条   | **已拒绝** | **易受攻击。** 通常是经过简单修改的常见词汇。 易受标准猜测攻击。  |
| **中等** | 黄绿色条  | **已接受** | **适中。** 可提供针对无限速在线攻击的基本保护。           |
| **强**  | 浅绿色条  | **已接受** | **高。** 难以猜测。 可抵御大多数攻击和标准破解尝试，提供可靠保护。 |
| **安全** | 绿色条   | **已接受** | **最高。** 数学上复杂且极难预测。 可抵御复杂攻击，提供最佳保护。  |

<Aside type="tip">
  要获得**安全**评级，我们建议使用由多个不相关词汇组成的密码短语，或使用密码管理器生成的长随机字符串。
</Aside>

### 安全密钥和通行密钥

在**安全密钥和通行密钥**部分，您可以注册实体安全密钥或通行密钥，为您的账户添加额外保护层。 此密钥与您的密码配合使用，在登录时验证您的身份。

### 身份验证 App

在**身份验证 App** 部分，您可以启用双重身份验证 (2FA) 为您的账户添加额外安全层。 在您的移动设备或电脑上使用身份验证器应用生成一次性验证码。

要启用两步验证，请按以下步骤操作：

<Steps>
  1. 打开您的**账户设置**，然后转到**账户**选项卡。
  2. 在**身份验证 App** 部分点击**启用**。
  3. 使用您移动设备上的 **Google 身份验证器**应用，扫描屏幕上的二维码。
  4. 输入您的身份验证 App 生成的 6 位验证码，然后点击**下一步**。 <Image src={twoFactorAuthenticationQRCode} alt="身份验证 App" class="width-md" />
  5. 下载恢复码，以便在无法访问您的移动设备时使用。 <Image src={twoFactorAuthenticationRecoveryCodes} alt="恢复码" class="width-md" />
</Steps>

### 重置双重身份验证

如果您无法访问您的身份验证 App 和恢复码，您可以请求重置 2FA 以重新访问您的账户。

要请求重置 2FA，请按照以下步骤操作：

<Steps>
  1. 在 Crowdin 登录页面，输入您的电子邮件（或用户名）和密码。
  2. 在**双重身份验证**界面，点击**请求重置双重身份验证**。
  3. 您将被重定向到**重置 2FA 方式**页面。 要验证您的身份，请尽可能提供多种验证因素：
     * **设备验证：** 此步骤为自动验证。 Crowdin 会检查您是否曾使用此浏览器登录您的账户。
     * **验证电子邮件地址：** 点击**发送验证码**，检查您的电子邮件获取验证码，然后输入。
     * **验证个人访问令牌：** 如果您保存了账户中的个人访问令牌 (PAT)，请将其粘贴到字段中。
  4. 提供可用的验证因素后，点击**提交**。
</Steps>

您的重置请求将被发送至 Crowdin 支持团队进行审核。 您将收到一封确认您的请求已被接收的电子邮件。

<Aside>
  为保护您的账户免受欺诈性访问，在禁用 2FA 之前有强制性的**三个工作日**安全等待期。 此期限在我们的支持团队成功验证您的身份后开始计算。 出于安全目的，团队可能会联系您以请求提供其他详细信息来确认您的账户所有权。
</Aside>

<Aside type="tip">
  在提交重置请求之前，请仔细检查您是否保存了首次设置 2FA 时提供的恢复码。
</Aside>

### 新设备验证

启用**新设备验证**，在从新设备登录时添加额外的安全步骤。 Crowdin 将向您的注册电子邮件地址发送验证码，以在允许从新设备访问之前确认您的身份。 成功登录后，该设备将被添加到受信任设备列表中。

<Aside>
  仅当您使用任意 2FA，例如身份验证 App 或安全密钥时，才能禁用此选项。
</Aside>

启用此功能后，您将看到受信任设备列表。 您可以通过点击**移除**按钮从列表中移除任何设备。 或者，您可以通过点击**移除所有设备**来移除所有受信任设备。

### 身份验证提供商

在**身份验证提供商**部分，您可以管理关联到您 Crowdin 账户的账户。 这些关联允许您使用 Google、Facebook、GitHub、X、GitLab 和 Crowdin 企业版等提供商一键登录。

### 会话

在**会话**部分，您可以查看已访问您 Crowdin 账户的设备列表，包括位置、最后活动日期、IP 地址和设备类型等详情。 撤销您不认识的单个会话，或通过点击**全部撤销**撤销除当前会话之外的所有会话。

<Image src={sessions} alt="会话" />

### Sudo 模式

Crowdin 在执行敏感操作，例如更改密码、管理电子邮件或身份验证设置以及删除账户之前，会要求您确认密码。 确认后，您可以在接下来的五分钟内执行这些操作，无需重新进行身份验证。

<Image src={sudoModeConfirmingCredentials} alt="确认凭据" class="width-md" />

## 通知

在**通知**选项卡中，您可以自定义您接收的通知以及通知的发送方式。 这包括设置不同通知渠道的偏好以及为特定项目配置规则。

* 应用内（Crowdin）

## API

在 **API** 选项卡中，您可以创建在使用 Crowdin API 时用于授权的个人访问令牌，并查看您账户的 API 调用历史记录。

### 个人访问密钥

个人访问令牌作为密码的替代方案，用于在 Crowdin 中授权第三方应用程序和脚本。 令牌列表显示每个令牌的**名称**、部分遮蔽的**令牌**、**创建**日期、**最后使用**信息以及**到期**时间。 如果令牌设置了到期日期，到期后将变为非活动状态，但仍会保留在列表中供参考。

<Image src={tokens} alt="个人访问密钥" />

<Aside type="caution">
  请像对待密码一样对待个人访问令牌，并妥善保管。 使用令牌作为环境变量，而不是将其硬编码到脚本中。
</Aside>

### 用例

在以下情况下，您需要使用个人访问令牌进行授权：

* 通过 [Crowdin API](/zh/developer/api/) 自动化本地化工作流程。
* 使用 [Crowdin CLI](https://crowdin.github.io/crowdin-cli/) 管理和同步本地化资源。
* 从设计工具（如 Figma、Sketch、Adobe XD）推送设计内容以进行翻译。
* 使用 IDE 插件（如 Visual Studio Code、Android Studio）上传和下载内容。
* 以及其他需要安全访问您 Crowdin 账户的集成或工具。

### 创建个人访问令牌

创建新的个人访问令牌时，您可以为其命名以便日后识别，设置到期日期，选择[特定作用域](/developer/understanding-scopes/)，并在需要时使用**细粒度访问**选项限制所选作用域的资源可见性。 例如，您可以创建一个仅与特定项目交互而无权访问其他项目的令牌。 因此，当发出 API 请求以检索所有项目列表时，只会返回该选定项目。

要创建新的个人访问令牌，请遵循以下步骤：

<Steps>
  1. 打开您的**账户设置**，然后转到 **API** 选项卡。
  2. 点击**新建令牌**。
  3. 在弹出的对话框中，输入名称以便日后识别该令牌。
  4. 为令牌设置到期日期。 默认情况下，**到期**设置为**永不**。
  5. 选择所需的作用域。
  6. *（可选）* 要限制对特定资源的访问，请点击**精细化访问**并选择所需的范围和相关资源。
  7. 单击 **创建**。
</Steps>

创建新令牌后，请务必立即复制并保存。 出于安全原因，该令牌不会再次显示。 您可以根据需要创建任意数量的个人访问令牌。

<Aside type="caution">
  如果在创建令牌时未选择某个资源，尝试通过 ID 访问该资源将导致 <code style="white-space: nowrap">404 Not Found</code> 错误。
</Aside>

### 撤销个人访问令牌

如果个人访问令牌不再需要或您怀疑其已泄露，请将其撤销。

要撤销个人访问令牌，请遵循以下步骤：

<Steps>
  1. 打开您的**账户设置**，然后转到 **API** 选项卡。
  2. 在列表中找到该令牌，然后点击其旁边的**撤销**以移除其访问权限。
</Steps>

### 调用历史

在**调用历史记录**部分，您可以查看与您 Crowdin 账户关联的 API 调用列表。 使用过滤器查看全部、成功或失败的调用，并选择特定操作组或其中的单个操作。 **应用程序名称**字段允许您搜索与特定应用程序相关的调用。

## 应用

在**应用**选项卡中，管理您 Crowdin 账户中安装的 Crowdin 应用。 您可以查看和管理当前已安装的应用，搜索特定应用，并使用每个应用上的**编辑**或**卸载**选项。

默认情况下，已安装的应用按安装日期排序，最近安装的应用显示在最前面。 您也可以使用可用的表格列排序选项对应用进行排序。

单击**从商店安装**，添加由 Crowdin 和其他开发者在 Crowdin [商店](https://store.crowdin.com/)中开发的应用。 或者，点击**安装私有应用**以手动安装您自己开发的自定义应用。

## 机器人

管理与 Crowdin 交互的自定义应用的机器人。 机器人允许应用程序在平台内代表您执行特定操作。 您可以从 [Crowdin 商店](https://store.crowdin.com/tags/agent)添加机器人，并根据需要管理其对项目和资源的权限。

## 安全日志

**安全日志**选项卡允许您跟踪 Crowdin 账户发生的重要事件（包括事件类型、使用的设备、IP 地址和日期）。

安全日志包括诸如登录、密码和用户名更改等事件。

## Webhook

在 **Webhooks** 选项卡中，您可以配置账户 webhook 以接收有关您 Crowdin 账户中发生的关键事件的通知。 设置完成后，当这些事件发生时，Crowdin 将通过 HTTP 向指定的 webhook URL 发送包含数据的 POST 或 GET 请求。

您可以为以下事件类型创建账户 webhook：

* 项目已创建
* 项目已删除

## OAuth

在 **OAuth** 选项卡中，您可以创建 OAuth 应用程序，该应用程序可用于向 Crowdin API 发出授权请求或作为单点登录服务，并管理您已授权使用您账户的已连接 OAuth 应用。

该功能的一些最常见用途包括：

* OAuth 应用允许您向 [Crowdin API](/zh/developer/api/) 发出授权请求。
* OAuth 应用经常用作单点登录服务。 您可以允许用户使用他们的 Crowdin 账户登录您的服务。

<ReadMore>
  请参阅[授权 OAuth 应用程序](/developer/authorizing-oauth-apps/)，了解如何授权 OAuth 应用程序访问您的 Crowdin 账户。
</ReadMore>

### 添加新应用

要添加新应用，请遵循以下步骤：

<Steps>
  1. 在右上角，点击您的个人头像并选择**设置**。
  2. 切换到 **OAuth** 选项卡并点击**新建应用程序**。
  3. 在弹出的对话框中，指定以下信息：
     * **名称**和**描述** &ndash;（可选）将在用户授权应用程序访问其 Crowdin 账户时向他们显示。
     * **授权回调 URL** &ndash; 用户使用 Crowdin 授权后将被发送到的 URL。 您可以添加多个 URL，以逗号分隔（无需使用引号）。
     * 从可用的[范围](/zh/developer/understanding-scopes/)列表中选择您的应用所需的访问权限。
  4. 单击 **创建**。 <Image src={oauthAppsNewApp} alt="创建 OAuth 应用" class="width-md crowdin-modal" />
</Steps>

<Aside>
  默认情况下，新创建的 OAuth 应用仅限于应用所有者的 Crowdin 项目。 要使它们可在整个 Crowdin 范围内使用，请通过相应请求[联系支持团队](https://crowdin.com/contacts)。
</Aside>

### 修改 OAuth 应用程序

创建 OAuth 应用程序后，您可以对其进行更改。 前往您的账户**设置 > OAuth 应用程序**，查看您账户下创建的 OAuth 应用列表、应用创建日期以及每个应用的使用用户数。

要查找特定应用程序，请在列表顶部的搜索字段中输入其名称或客户端 ID。

在必要应用的*操作*列中点击 <Icon name="mdi:dots-horizontal" class="inline-icon" /> 以打开上下文菜单。

使用可用选项，您可以执行以下操作：

* **编辑** &ndash; 更新应用程序名称、描述、URL 和范围。 访问所创建应用程序的客户端 ID 和客户端密钥。
* **重置密钥** &ndash; 重置应用程序的客户端密钥。
* **撤销令牌** &ndash; 撤销所有用户令牌。
* **删除** &ndash; 删除应用程序。

### 管理已连接的 OAuth 应用程序

在**已连接的 OAuth 应用程序**部分，您可以查看您已授权使用您账户的已连接 OAuth 应用，并根据需要撤销访问权限。

## 我的费率

在**我的费率**选项卡中，您可以创建和管理报告设置的个人模板。 这些模板仅在您的个人资料中可见，可用于报告。

每个模板可以包括：

* **基础费率** – 设置完整翻译和校对的费率。 默认应用于所有语言。
* **净费率方案** – 根据完整翻译费率，为翻译记忆、MT、AI 及其他匹配类型设置百分比费率。
* **自定义费率** – 添加特定语言（在某些情况下，也可针对特定用户）的费率配置。
* **附加选项** – 包含或排除自动翻译的字符串，调整重复翻译的分类方式，并计算内部模糊匹配。

保存的模板有助于标准化您的费率设置，使配置可在报告中轻松复用。 要使用已保存的模板，请在配置报告时点击**模板**，然后选择您要应用的模板。

要创建新的费率模板，请按以下步骤操作：

<Steps>
  1. 前往**账户设置 > 我的费率**。
  2. 点击**新建**。
  3. 输入模板名称并选择指标配置。
  4. 定义基本费率并\*（可选）\*添加自定义费率。
  5. 为匹配类别配置净费率方案。
  6. \*（可选）\*为自动翻译字符串和内部模糊匹配配置其他选项。
  7. 点击**保存**。
</Steps>

要管理现有模板，请使用**操作**列中的**编辑**或**删除**。

## 自定义导航

在 **自定义导航** 标签页中，您可以管理个人资料导航项的显示方式，包括其顺序和可见性。 The list includes both default navigation sections and any installed apps that add a navigation item。 配置的顺序和可见性适用于两种布局：垂直左侧边栏（菜单固定时）和页面顶部的水平标签页（菜单未固定时）。

<ReadMore>
  阅读更多关于在[个人资料页面](/zh/for-translators/#个人资料页面)上管理菜单布局的内容。
</ReadMore>

### 重新排列导航项

要重新排列导航项，请将鼠标悬停在要移动的项目旁边的 <Icon name="material-symbols:drag-indicator" class="inline-icon" /> 上，然后单击并将其拖动到列表中的目标位置。

### 切换项目可见性

要隐藏或显示导航项，请清除其名称旁边的复选框以将其隐藏，或勾选复选框以使其可见。

<Aside>
  **项目** 项始终显示，无法隐藏。 隐藏项目中的内容仍然安全且可访问。
</Aside>

### 重置导航设置

要将导航顺序和可见性重置为默认设置，请滚动到列表底部，然后单击 **重置为默认值**。
