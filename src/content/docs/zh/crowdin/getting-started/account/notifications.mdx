Crowdin 提供多种类型的通知，以满足用户的不同需求，例如：

* *项目更新*：通知用户项目的变更，例如新增待翻译内容或对现有内容的更新。
* *翻译活动*：通知用户新的翻译、审批或翻译评论。
* *任务管理*：这些通知告知用户新任务、截止日期和任务完成情况。
* *团队沟通*：这些提醒通知用户项目内的消息和讨论。

要查看通知，请点击页面右上角的铃铛图标 <Icon name="mdi:bell-outline" class="inline-icon" />。

## Slack 集成

通过直接发送到 Slack 的更新，您会立即知道您管理或贡献的 Crowdin 项目中发生了什么。 在 Crowdin 账户与 Slack 整合后，选择您想接收的通知，Crowdin 机器人将把它们作为私信发送。

<Aside>
  必须为每个希望在 Slack 中接收通知的用户单独配置 Crowdin 的 Slack 集成。
</Aside>

要通过 Slack 接收 Crowdin 通知，请遵循以下步骤：

<Steps>
  1. 点击右上角的个人头像，然后选择**设置**。
  2. 切换到**通知**选项卡，然后点击**连接 Slack**。 <Image src={slackConnect} alt="连接 Slack" />
  3. 在 Slack 端授权与 Crowdin 的连接。 <Image src={slackAuthorize} alt="授权 Slack" class="width-lg" />
  4. 返回 Crowdin 并选择您想要在 Slack 中接收的通知。
</Steps>

<ReadMore>
  阅读更多关于[全局通知设置](/account-settings/#notifications)的信息。
</ReadMore>

您将以私信的形式从 Crowdin 机器人收到所选类型的通知。

<Image src={slackCrowdinNotifications} alt="Slack 中的 Crowdin 通知" />

您可以随时禁用 Crowdin 通知并断开 Slack 连接。

## 自定义通知

自定义通知允许您接收在 Crowdin 发生的新事件的信息。 一旦您为您的帐户配置了自定义通知，Crowdin 将开始通过 HTTP 向自定义通知 URL 发送带有数据的 POST 请求。

### 用例

您可以配置自定义通知以构建与服务或后端的集成。 例如：

* 设置自定义通知以将通知发送到您使用的系统。
* 将信息传递给第三方服务，并附有具体的要求（如内容类型、标题、有效载荷）。
* 创建与 Crowdin 的自定义集成。

### 配置

要在 Crowdin 中配置自定义通知，请按照以下步骤操作：

<Steps>
  1. 打开您的**账户设置**，然后转到**通知**选项卡。
  2. 在**通知自定义渠道**部分，点击**设置通知自定义渠道**开始配置您的自定义通知。
</Steps>

您需要提供以下信息来设置自定义通知：

* 应发送回调的 URL 地址。
* POST 请求方法的内容类型（`multipart/form-data`、`application/json` 或 `application/x-www-form-urlencoded`）。

您可以在您的自定义通知中添加特殊标题（可选）。 作为一种授权方法，它们可以用来提供更多的安全保障。 例如，如果您添加了标题，您的自定义通知端点可以验证它们，并确保来自 Crowdin 的信息。

配置自定义通知时，点击**测试通知**以查看您的应用程序对该调用的反应。 配置完成后，点击**保存**。

<Image src={customNotificationsSetUp} alt="设置自定义通知" class="width-xl" />

取决于您对自定义通知管理的方法，您可能需要添加 Crowdin IP 地址到防火墙，允许 Crowdin 打开预配置的自定义通知 URL。

<ReadMore>
  详细了解 [IP 地址](/developer/ip-addresses/)。
</ReadMore>

### 为 Discord 配置 {#discord}

要在 Discord 中设置来自 Crowdin 的自定义通知，请按照以下步骤操作。

#### 创建 Discord 账户

如果您还没有 Discord 账户，则需要[创建一个](https://discord.com/register)。

<Aside>
  您只能通过 Discord 桌面应用程序或 Discord 网页界面配置来自 Crowdin 的自定义通知。 配置完成后，您还可以通过 Discord 移动应用程序接收这些通知。
</Aside>

下载并安装 [Discord 桌面应用](https://discord.com/download)。 或者，您可以使用 Discord 网络界面进行进一步配置。

#### 创建 Discord Webhook

要在 Discord 中接收通知，您需要创建一个 Webhook。 完成后，复制 Webhook URL。 您需要在 Crowdin 端进行配置。

<ReadMore>
  阅读更多关于[创建 Discord Webhook](https://support.discord.com/hc/en-us/articles/228383668-Intro-to-Webhooks) 的信息。
</ReadMore>

#### 配置自定义通知频道 {#custom-channel-discord}

<Steps>
  1. 打开您的**账户设置 > 通知**。
  2. 点击**设置自定义通知渠道**。
  3. 在弹出的对话框中，将 Discord Webhook URL 粘贴到 **URL** 字段中。
  4. 为**内容类型**选择 **application/json**。
  5. 将以下负载粘贴到**负载**字段中：
     ```json
     {
       "content": "{{notification-message}}"
     }
     ```
  6. 点击**测试通知**即可在您的 Discord 频道接收来自 Crowdin 的测试消息。
  7. 配置完成后，点击**保存**。
</Steps>

### 为 Telegram 配置 {#telegram}

要在 Telegram 中设置来自 Crowdin 的自定义通知，请按照以下步骤操作。

#### 创建 Telegram 账户

如果您没有预先注册 Telegram 账号，您需要连接一个电话号码。 Telegram 使用您的电话号码作为首要凭据，您将使用它登录 Telegram 账号。

因此，首先请使用 [Android](https://telegram.org/dl/android) 或 [iOS](https://telegram.org/dl/ios) 下载并安装 Telegram 应用。
注册后，您可以使用 PC 或 Mac 上的 [Telegram 网页界面](https://telegram.org/dl/webogram) 来简化后续配置，这部分由您自己决定。

#### 创建 Telegram 机器人

要在 Telegram 中接收通知，您需要创建机器人。 一旦完成，您将获得一个机器人 API 令牌。 请务必保存好令牌，稍后会需要它。

<ReadMore>
  阅读更多关于[创建 Telegram 机器人](https://core.telegram.org/bots#creating-a-new-bot)的信息。
</ReadMore>

#### 配置 Telegram 频道

下一步是创建一个公开的 Telegram 频道，用于接收来自 Crowdin 的通知。 稍后您可以根据需要更改频道的隐私设置。

要创建一个 Telegram 频道，请遵循以下步骤：

<Steps>
  1. 在您的设备上打开 Telegram。
  2. 点击笔形图标并选择**新建频道**。
  3. 在**频道名称**字段中为您的频道命名。
  4. *（可选）* 指定频道描述。
  5. *（可选）* 点击相机图标为您的频道设置显示图片。
  6. 点击右上角的**下一步**。
  7. 选择**公共**频道类型。
  8. 为您的频道指定永久链接。 别人搜索和加入到您的频道中将需要这个链接。
  9. 点击右上角的**下一步**。
  10. *（可选）* 在此步骤中，Telegram 会要求您向您的 Telegram 频道添加订阅者。 选择您想要添加的联系人，包括您的新机器人（为您的机器人提供管理员权限）。 您可以选择暂时不添加任何成员，因为您稍后可以这样做。
  11. 点击右上角的**下一步**继续，并在 Telegram 上创建您的频道。
</Steps>

#### 配置自定义通知频道 {#custom-channel-telegram}

<Steps>
  1. 打开您的**账户设置**，然后转到**通知**选项卡。
  2. 点击**设置自定义通知渠道**。
  3. 在弹出的对话框中，按以下格式指定 **URL**：`https://api.telegram.org/bot{bot_API_token}/sendMessage`。
  4. 为**内容类型**选择 **application/json**。
  5. 将以下负载粘贴到**负载**字段中：
     ```json
     {
       "chat_id": "{public_chat_id}",
       "text": "{{notification-message}}"
     }
     ```
     其中 `{public_chat_id}` &ndash; 您在 Telegram 频道配置期间为频道链接指定的公共聊天 ID。
  6. 点击**测试通知**即可在您的 Telegram 频道接收来自 Crowdin 的测试消息。
  7. 配置完成后，点击**保存**。
</Steps>

#### 更改 Telegram 频道类型

如果您想将您的 Telegram 频道设为私密，并继续接收来自 Crowdin 的通知，请按以下步骤操作：

<Steps>
  1. 首先，您需要通过访问以下链接获取 Telegram 频道的原始聊天 ID：`https://api.telegram.org/bot{bot_API_token}/sendMessage?chat_id={public_chat_id}&text=Test`
  2. 您将收到 JSON 格式的响应，其中包含您的 Telegram 频道的原始聊天 ID。
  3. 在 Crowdin 中，打开您的**账户设置**，然后转到**通知**选项卡。
  4. 点击**设置自定义通知渠道**。
  5. 在弹出的对话框中，将聊天 ID 替换为上面 JSON 响应中收到的聊天 ID，以修改 **URL**。
  6. 点击**保存**。
  7. 在 Telegram 中，打开您的频道。
  8. 点击您的频道名称，然后点击**编辑**。
  9. 点击**频道类型**。
  10. 选择**私密**并点击**完成**。
</Steps>

<Aside>
  请确保将 `{bot_API_token}` 和 `{public_chat_id}` 替换为您实际的 Telegram 机器人 API 令牌和公共聊天 ID。
</Aside>
