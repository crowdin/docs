XML 文件与 CSV 和 XLSX 文件类似，上传到项目后需要进行额外的配置，以便系统可以导入这些文件的内容。

要配置 XML 文件，请按照下列步骤操作：

<Steps>
  1. 打开您的项目并转到**原文 > 文件**。
  2. 单击文件旁边的 **配置**，打开配置窗口。
  3. 选择需要翻译的内容，然后单击 **保存并导入** 继续。
</Steps>

<Image src={xmlFileConfigure} alt="配置 XML 文件" />

<LinkCard title="使用 Crowdin 进行 XML 翻译" href="https://store.crowdin.com/xml" target="_blank" />

<Aside type="caution" title="限制">
  XML 文件在[基于字符串](/zh/enterprise/creating-project/#string-based-project)的项目中不受支持。
</Aside>

## 配置选项

配置 XML 文件时，您将有以下选项：**基本配置**和**高级配置**。 此外，您还可以使用**内容句段**功能，该功能适用于这两种配置选项。

### 基本配置

基本配置允许您选择所有可供翻译的节点和/或所有属性。

<Image src={xmlFileBasic} alt="XML 基本配置" />

### 高级配置

高级配置允许您选择可供翻译的特定节点和特定属性。

<Image src={xmlFileAdvanced} alt="XML 高级配置" />

文件结构通过图标来描述：

* <Icon name="mdi:label" class="inline-icon" /> – 节点
* <Icon name="mdi:tag" class="inline-icon" /> – 属性

<LinkCard title="适用于 XML 的 XSL 转换器" description="使用适用于 XML 的 XSL 转换器应用程序，直接在编辑器中预览您的 XML 文件。" href="https://store.crowdin.com/xslt-preview" target="_blank" />

### 同时配置所有 XML 文件

要一次配置所有 XML 文件，请单击**文件**页面顶部的**配置全部**。

使用**配置全部**时，请确保所有 XML 文件包含相同的结构元素。 否则，一些需要翻译的节点可能在配置期间不可用，并且不会导入到项目中。 在这种情况下，有必要单独配置每个 XML 文件。

在配置窗口中，您将能够在所选文件之间切换以预览并验证应用的配置是否对所有文件都正确。

## 内容句段

您可以使用**内容句段**将源文件的内容分割成较短的文字内容，主要是句子和短段落。 这有助于翻译包含若干句子的内容单元，因为翻译较小篇幅的文本更加容易些。 将使用 Segmentation Rules eXchange (SRX) 进行自动的内容分割。

<Aside>
  启用内容分段后，译文上传由实验性机器学习技术处理。
</Aside>
