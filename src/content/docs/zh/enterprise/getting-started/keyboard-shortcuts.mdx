您可以使用键盘快捷键在 Crowdin Enterprise 工作区中快速导航并执行常见操作，无需使用鼠标。

<Aside type="tip">
  要直接在界面中查看所有可用快捷键的完整列表，请点击右上角的 <Icon name="mdi:help-circle-outline" class="inline-icon" /> **帮助**图标并选择**键盘快捷键**，或直接按键盘上的 <KBD>Shift</KBD> + <KBD>/</KBD>（或 <KBD>?</KBD>）。
</Aside>

## 全局快捷键

使用这些快捷键导航工作区的主要部分并管理全局资源。

  <thead>
    <tr>
      <th>快捷键</th>

    </tr>
  </thead>
  <tbody>
    <tr>
      <td><KBD>c</KBD></td>
      <td>创建当前资源（例如，在工作区中创建新项目）</td>
    </tr>
    <tr>
      <td><KBD>Shift</KBD> + <KBD>h</KBD></td>
      <td>打开帮助聊天</td>
    </tr>
    <tr>
      <td><KBD>Shift</KBD> + <KBD>/</KBD> 或 <KBD>?</KBD></td>
      <td>显示键盘快捷键</td>
    </tr>
    <tr>
      <td><KBD>g</KBD> 然后 <KBD>w</KBD></td>
      <td>前往工作区</td>
    </tr>
    <tr>
      <td><KBD>g</KBD> 然后 <KBD>n</KBD></td>
      <td>前往通知</td>
    </tr>
    <tr>
      <td><KBD>g</KBD> 然后 <KBD>m</KBD></td>
      <td>前往消息</td>
    </tr>
    <tr>
      <td><KBD>g</KBD> 然后 <KBD>t</KBD></td>
      <td>前往任务</td>
    </tr>
    <tr>
      <td><KBD>g</KBD> 然后 <KBD>c</KBD></td>
      <td>
        <p>前往[客户端](/zh/enterprise/for-vendors/#clients)</p>
        <p><Aside>仅适用于供应商组织</Aside></p>
      </td>
    </tr>
    <tr>
      <td><KBD>g</KBD> 然后 <KBD>i</KBD></td>
      <td>
        <p>前往[传入项目](/zh/enterprise/clients/#incoming-projects)</p>
        <p><Aside>仅适用于供应商组织</Aside></p>
      </td>
    </tr>
    <tr>
      <td><KBD>g</KBD> 然后 <KBD>v</KBD></td>
      <td>前往供应商</td>
    </tr>
    <tr>
      <td><KBD>g</KBD> 然后 <KBD>u</KBD></td>
      <td>前往用户</td>
    </tr>
    <tr>
      <td><KBD>g</KBD> 然后 <KBD>s</KBD></td>
      <td>前往[组织设置](/zh/enterprise/organization-settings/)</td>
    </tr>
    <tr>
      <td><KBD>n</KBD></td>
      <td>分页下一页</td>
    </tr>
    <tr>
      <td><KBD>p</KBD></td>
      <td>分页上一页</td>
    </tr>
    <tr>
      <td><KBD>s</KBD></td>
      <td>在当前资源中搜索（例如，在工作区中搜索项目）</td>
    </tr>
  </tbody>
</table>
