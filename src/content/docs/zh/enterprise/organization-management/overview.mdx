一个组织可以将所有参与本地化工作的公司成员和贡献者（翻译员、审核员、供应商等） 集中到同一空间。

在您的组织内，您可以：

* 单独或作为群组的一部分创建和管理项目。
* 创建工作流模板并将其应用于单个项目和群组。
* 授予不同级别的管理员权限。
* 将翻译员、审核员和供应商分配到特定工作流步骤中。
* 使用[内置 AI 功能](/zh/enterprise/crowdin-ai/)提升翻译质量和工作效率。
* 通过 [Crowdin Store](https://store.crowdin.com/) 中的应用和集成扩展功能。

## 创建组织

在 Crowdin Enterprise 中创建组织时，您同时也会创建个人账户，并自动成为组织所有者。 所有者对组织级别设置、账单、用户管理等拥有完全访问权限。 其他用户[通过邀请加入现有组织](#通过邀请注册)并成为组织成员。

要创建组织并注册账户，请按照以下步骤操作：

<Steps>
  1. 打开注册页面。
  2. 选择您偏好的数据中心（**美国数据中心**或**欧盟数据中心**）。 这将决定您组织的数据（文件、字符串及其他资源）的存储位置。
  3. 输入您的组织名称。 该名称也将用于 URL（例如，`organization-name.crowdin.com`）。
  4. 使用您的账户信息填写所有字段，然后单击**创建组织**。 <Image src={createWorkspace} alt="创建组织" class="width-lg" />
  5. 前往您的收件箱，打开激活电子邮件，然后单击**激活组织**。
</Steps>

<Aside type="caution">
  URL 中使用的组织名称不可更改。 所有其他个人数据可在您的[账户设置](/zh/enterprise/account-settings/)中进行更改。
</Aside>

<Aside type="tip">
  如果您希望为组织使用自定义 URL，可以设置[自定义域名](/zh/enterprise/organization-settings/#自定义域名)。 这对于公开项目或众包活动尤为有用。
</Aside>

## 通过邀请注册

加入组织后，您可以与团队一起开始处理本地化项目。
收到邀请后，请按照以下步骤加入组织：

<Steps>
  1. 打开邀请电子邮件，然后单击**接受邀请**。
  2. 创建用户名和密码。
  3. 单击**创建账户**。 <Image src={acceptInvitation} alt="接受邀请" class="width-lg" />
</Steps>

## 登录到组织

组织创建并激活后，请按照以下步骤登录：

<Steps>
  1. 打开登录页。
  2. 输入您的组织名称，然后单击**继续**。 <Image src={logIn} alt="登录表单" class="width-lg" />
  3. 输入您的电子邮件或用户名和密码。
  4. 单击**登录**。
</Steps>

## 查找您的组织 URL

以下是找到您有权访问的组织的最快方法：

<Steps>
  1. 打开登录页面并选择 **Crowdin Enterprise**。
  2. 单击**忘记 URL？**。
  3. 添加您的电子邮箱地址。
  4. 单击**确认**。
</Steps>

我们将向您输入的地址发送与该地址关联的 Crowdin Enterprise 组织的登录链接。
