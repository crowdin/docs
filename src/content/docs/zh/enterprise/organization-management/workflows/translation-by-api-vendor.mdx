通过 API 供应商进行翻译是一个工作流步骤，用于向通过 API 与 Crowdin Enterprise 集成的供应商订购翻译。 它通过支持手动或定时下单来帮助实现翻译流程自动化，翻译结果将直接交付到您的项目。

</CardGrid>

## 将 API 供应商添加到组织

要使用此工作流步骤，请先向您的组织添加受支持的 API 供应商（例如 Gengo）。

<Steps>
  1. 打开您组织的**工作区**，然后在左侧栏中选择**商店**。
  2. 切换到 **Vendors** 选项卡，然后单击 Gengo 对应的 **Invite**。
  3. 在弹出的对话框中，保持选中 **Vendor as organization**，然后单击 **Invite**。 <Image src={storeInviteVendor} alt="邀请供应商对话框" />
  4. 返回您的 **Workspace**，在左侧边栏中选择 **Vendors**，然后单击 **Gengo**。
  5. 单击 **Set up account**。
  6. 在弹出的对话框中，粘贴您在 Gengo [Account settings > API settings](https://gengo.com/account/api_settings/) 中获取的 Gengo API 密钥，然后单击 **Done**。
</Steps>

## 将 API 供应商翻译步骤添加到您的工作流

您可以在工作流编辑器或工作流模板编辑器中添加*通过 API 供应商进行翻译*步骤。 在步骤设置中，从 *Vendor* 下拉菜单中选择供应商（例如 Gengo）。 如有需要，可调整 *Expertise*、*Tone* 和 *Purpose* 等附加选项以满足您的需求。

<Image src={workflowEditor} alt="工作流编辑器中的 API 供应商翻译" />

## 从 API 供应商订购翻译

将步骤添加到项目工作流后，项目的 **Workflow** 部分将出现 *API Vendor* 选项卡。

要订购翻译，请按照以下步骤操作：

<Steps>
  1. 打开您的项目并前往 **Workflow > API Vendor**。
  2. 选择目标语言。
  3. 单击 **Create order**。 <Image src={createOrder} alt="创建订单" />
  4. *（可选）* 如果您希望定期自动发送翻译订单而无需手动操作，请启用 **Automatically order translations (every hour)**。
</Steps>

供应商完成翻译后，译文将自动交付回项目。

## 订单历史记录

要查看历史订单，请在 **Workflow > API Vendor** 页面向下滚动至 **Orders history** 部分。 该部分显示每个订单的语言、字数、状态和价格。 如需查看更详细的订单信息，请单击相应的订单条目。

<Image src={ordersHistory} alt="订单历史记录" class="width-2xl" />

您也可以前往 **Workspace > Vendors** 并选择所需供应商（例如 Gengo），查看所有项目的订单。 在 **Orders** 选项卡上，使用可用的筛选条件（例如 Project、Language、Status、Created）缩小结果范围并查看相关订单详情。
