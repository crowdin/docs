You can use keyboard shortcuts to quickly navigate through your Crowdin Enterprise workspace and perform common actions without using a mouse.

<Aside type="tip">
  To view the full list of available shortcuts directly in the UI, click the <Icon name="mdi:help-circle-outline" class="inline-icon" /> **Help** icon in the upper-right corner and select **Keyboard Shortcuts**, or simply press <KBD>Shift</KBD> + <KBD>/</KBD> (or <KBD>?</KBD>) on your keyboard.
</Aside>

## Global Shortcuts

Use these shortcuts to navigate the main sections of your workspace and manage global resources.

<table>
  <thead>
    <tr>
      <th>Shortcut</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><KBD>c</KBD></td>
      <td>Create current resource (e.g., a new project when in the Workspace)</td>
    </tr>
    <tr>
      <td><KBD>Shift</KBD> + <KBD>h</KBD></td>
      <td>Open help chat</td>
    </tr>
    <tr>
      <td><KBD>Shift</KBD> + <KBD>/</KBD> or <KBD>?</KBD></td>
      <td>Display keyboard shortcuts</td>
    </tr>
    <tr>
      <td><KBD>g</KBD> then <KBD>w</KBD></td>
      <td>Go to Workspace</td>
    </tr>
    <tr>
      <td><KBD>g</KBD> then <KBD>n</KBD></td>
      <td>Go to Notifications</td>
    </tr>
    <tr>
      <td><KBD>g</KBD> then <KBD>m</KBD></td>
      <td>Go to Messages</td>
    </tr>
    <tr>
      <td><KBD>g</KBD> then <KBD>t</KBD></td>
      <td>Go to Tasks</td>
    </tr>
    <tr>
      <td><KBD>g</KBD> then <KBD>c</KBD></td>
      <td>
        <p>Go to [Clients](/enterprise/for-vendors/#clients)</p>
        <p><Aside>Only for Vendor organizations</Aside></p>
      </td>
    </tr>
    <tr>
      <td><KBD>g</KBD> then <KBD>i</KBD></td>
      <td>
        <p>Go to [Incoming projects](/enterprise/clients/#incoming-projects)</p>
        <p><Aside>Only for Vendor organizations</Aside></p>
      </td>
    </tr>
    <tr>
      <td><KBD>g</KBD> then <KBD>v</KBD></td>
      <td>Go to Vendors</td>
    </tr>
    <tr>
      <td><KBD>g</KBD> then <KBD>u</KBD></td>
      <td>Go to Users</td>
    </tr>
    <tr>
      <td><KBD>g</KBD> then <KBD>s</KBD></td>
      <td>Go to [Organization Settings](/enterprise/organization-settings/)</td>
    </tr>
    <tr>
      <td><KBD>n</KBD></td>
      <td>Pagination next</td>
    </tr>
    <tr>
      <td><KBD>p</KBD></td>
      <td>Pagination previous</td>
    </tr>
    <tr>
      <td><KBD>s</KBD></td>
      <td>Search in a current resource (e.g., projects when in the Workspace)</td>
    </tr>
  </tbody>
</table>
