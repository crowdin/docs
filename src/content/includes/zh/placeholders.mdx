| **名称**                     | **描述**                                   |
| -------------------------- | ---------------------------------------- |
| `%original_file_name%`     | 原始文件名                                    |
| `%original_path%`          | 获取 Crowdin 项目中的父文件夹名称，以构建生成包中的文件路径       |
| `%file_extension%`         | 原始文件扩展名                                  |
| `%file_name%`              | 不含有扩展名的文件名                               |
| `%language%`               | 语言名称（例如：Ukrainian）                       |
| `%two_letters_code%`       | ISO 639-1 语言代码（例如，uk）                    |
| `%three_letters_code%`     | ISO 639-2/T 语言代码（例如，ukr）                 |
| `%locale%`                 | 语言区域（例如，uk-UA）                           |
| `%locale_with_underscore%` | 语言区域（例如：uk_UA）                           |
| `%android_code%`           | 用于命名 "values-" 目录的 Android 语言区域标识符       |
| `%osx_code%`               | 用于命名 ".lproj" 目录的 OS X 本地化标识符            |
| `%osx_locale%`             | 用于命名翻译资源的 OS X 语言区域（例如：uk、zh-Hans、zh_HK） |
