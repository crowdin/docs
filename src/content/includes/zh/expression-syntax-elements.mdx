使用以下元素配置口语化表达式和自定义占位符。

| **元素**         | **别名**                         | **描述**                              | **用法示例**             |
| -------------- | ------------------------------ | ----------------------------------- | -------------------- |
| `start`        | `startofline`, `start of line` | 带有 ^ 标记的表达式                         |                      |
| `end`          | `end of`, `endofline`          | 带有 $ 标记的表达式                         |                      |
| `then`         | `find`                         | 向表达式中添加字符串                          | `then "abc"`         |
| `maybe`        |                                | 定义可能出现一次或不出现的字符串                    | `maybe "abc"`        |
| `word`         |                                | 匹配任何包含 a-z、A-Z、0-9，包括 _ (下划线) 字符的单词 |                      |
| `anything`     |                                | 接受任何字符串                             |                      |
| `range`        |                                | 向表达式中添加范围                           | `range "a,z,0,9"`    |
| `something`    |                                | 接受任何非空字符串                           |                      |
| `anythingbut`  | `anything but`                 | 接受除指定字符以外的任何字符串                     | `anythingbut "abc"`  |
| `anyof`        | `any`, `any of`                | 任何列出的字符                             | `anyof "abc"`        |
| `somethingbut` | `something but`                | 除这些字符以外的任何非空内容                      | `somethingbut "abc"` |
| `limit`        |                                | 添加字符限制                              | `limit "1,3"`        |
| `linebreak`    | `line break, br`               | 匹配 \r \n (可能只与其他元素一起使用)             |                      |
| `tab`          |                                | 匹配制表符 \t (可能只与其他元素一起使用)             |                      |
| `multiple`     |                                | 添加多个修饰符                             |                      |
