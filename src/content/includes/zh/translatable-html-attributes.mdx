当与基于 HTML 格式的文件一同工作时，视文件结构不同，某些 HTML 属性值可能视为可翻译的，而另一些不可翻译。 您可以在下表看到一些可翻译属性值的状态。

| **属性**        | **详细信息**                                                                                                       | **示例**                                                        |
| ------------- | -------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------- |
| `title`       | 如果包含在任意 HTML 标签中，可翻译                                                                                           | `<p title="value">Text</p>`                                   |
| `src`         | 如果包含在 `img`、`iframe`、`embed`、`video`、`audio`、`source`\*、`track`\* 标签中可翻译<br /> \* – 如果嵌套在父标签 `video`、`audio` 中 | `<audio src="sound.ogg" controls>Text</audio>`                |
| `href`        | 如果包含在 `a` 标签中，可翻译                                                                                              | `<a href="https://www.site.com">Site name</a>`                |
| `data`        | 如果包含在 `object` 标签中，可翻译                                                                                         | `<object data="image.jpg"></object>`                          |
| `value`       | 如果包含在 `input`、`button` 标签中，可翻译                                                                                 | `<input value="Text">`                                        |
| `placeholder` | 如果包含在 `input`、`textarea` 标签中，可翻译                                                                               | `<textarea placeholder="Enter some text here..."></textarea>` |
| `alt`         | 如果包含在 `img` 标签中，可翻译                                                                                            | `<img src="image.jpg" alt="Image">`                           |
| `label`       | 如果包含在 `optgroup`、`track`\* 标签中可翻译<br /> \* – 如果嵌套在父标签 `video`、`audio` 中                                        | `\<optgroup label="Text">`                   |
| `content`     | 如果包含在 `meta` 标签中，可翻译                                                                                           | `<meta name="description" content="Text">`                    |
