Use the elements below to configure verbal expressions and custom placeholders.

| **Element**    | **Aliases**                    | **Description**                                                                         | **Usage examples**   |
| -------------- | ------------------------------ | --------------------------------------------------------------------------------------- | -------------------- |
| `start`        | `startofline`, `start of line` | markér udtryk med ^                                                                     |                      |
| `end`          | `end of`, `endofline`          | markér udtrykket med $                                                                  |                      |
| `then`         | `find`                         | tilføj en streng til udtrykket                                                          | `then "abc"`         |
| `maybe`        |                                | definere en streng, der kan fremkomme én gang eller ikke                                | `maybe "abc"`        |
| `word`         |                                | matche ethvert ord indeholdende tegn fra a-z, A-Z, 0-9, inkl. _-tegnet (understregning) |                      |
| `anything`     |                                | acceptér enhver streng                                                                  |                      |
| `range`        |                                | føj et interval til udtrykket                                                           | `range "a,z,0,9"`    |
| `something`    |                                | acceptér enhver ikke-tom streng                                                         |                      |
| `anythingbut`  | `anything but`                 | acceptér enhver streng bortset fra det angivne tegn                                     | `anythingbut "abc"`  |
| `anyof`        | `any`, `any of`                | ethvert af de anførte tegn                                                              | `anyof "abc"`        |
| `somethingbut` | `something but`                | alt ikke-tomt, bortset fra disse tegn                                                   | `somethingbut "abc"` |
| `limit`        |                                | tilføj tegngrænse                                                                       | `limit "1,3"`        |
| `linebreak`    | `line break, br`               | match \r \n (må kun bruges med andre elementer)                                         |                      |
| `tab`          |                                | match faner \t (må kun bruges med andre elementer)                                      |                      |
| `multiple`     |                                | adds the multiple modifier                                                              |                      |
